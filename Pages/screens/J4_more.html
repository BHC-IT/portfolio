Optimise and automate distribution.

Our goal is to normalize the use of distribution by simplifying it.

We will add a paquet manager to C++.

Build with TypeScript and mongoDB, the API offer a fast and scalable way to sign-in user. OAuth2 is currently the only protocol supported, but more are to come.

Tests, linting, deployment and feedback are entirely automated with GitHub actions. Regression or potential failure can't go unnoticed.

A configuration file can be given to set-up each node in a specific way. Furthermore, most features can have their rules overloaded by another service for specific needs.
